## LINE Official Account Manager での設定手順

1. https://manager.line.biz にログイン → 対象アカウントを選択
2. 左メニュー「トークルーム管理」→「リッチメニュー」→「作成」
3. **表示設定**
   - タイトル: 任意(例: メインメニュー)
   - 表示期間: 開始日のみ設定(終了日は空欄でOK)
   - メニューバーのテキスト: `Menu / メニュー` がおすすめ(インバウンド向け)
   - デフォルト表示: オン
4. **コンテンツ設定**
   - テンプレート: 「大」→ 6分割(3列×2行)を選択
   - 「画像をアップロード」で `richmenu.png` を選択
5. 各エリアにアクションを設定(下記参照)→「保存」

## 各ボタンのアクション設定(推奨)

| 位置 | ボタン | アクション |
|---|---|---|
| 上段・左 | RESERVE ご予約 | テキスト: `予約希望 / I'd like to make a reservation` (トークで予約対応する場合) |
| 上段・中 | SCHEDULE 出勤一覧 | リンク: `https://sakuraspa-osaka.com/ja#schedule` ※出勤ページ実装後に差し替え |
| 上段・右 | PRICE 料金・コース | リンク: `https://sakuraspa-osaka.com/ja#price` |
| 下段・左 | THERAPISTS セラピスト一覧 | リンク: `https://sakuraspa-osaka.com/ja#therapists` ※実装後に差し替え |
| 下段・中 | ACCESS アクセス・営業時間 | リンク: `https://sakuraspa-osaka.com/ja#access` |
| 下段・右 | CONTACT お問い合わせ | テキスト: `お問い合わせ / Contact` またはリンク: `https://sakuraspa-osaka.com/ja#contact` |

※ インバウンド客はLINEのトーク内で完結させたい人が多いので、RESERVEは「テキスト送信」にして
トークで受けるのがおすすめ。スタッフ対応が難しければ予約フォームのURLに変更してください。

# 求人用リッチメニュー(セラピスト募集)

## ファイル

| ファイル | 内容 |
|---|---|
| `richmenu-recruit.png` | 求人用画像(2500×1686px、728KB) |
| `richmenu-recruit.html` | デザイン元データ |

対象は求職中の女性なので、白×桜ピンクの明るいトーン・日本語メイン。
各ボタン上部に訴求バッジ(未経験大歓迎・高バック保証・自由出勤など)入り。

## 各ボタンのアクション設定(推奨)

求人はトークでのやり取りが基本なので、テキスト送信中心がおすすめ:

| 位置 | ボタン | アクション |
|---|---|---|
| A(上段・左) | 応募する | テキスト: `応募希望です` |
| B(上段・中) | 給与・報酬 | テキスト: `給与・報酬について教えてください` |
| C(上段・右) | 待遇・働き方 | テキスト: `待遇・働き方について教えてください` |
| D(下段・左) | 体験入店 | テキスト: `体験入店を希望します` |
| E(下段・中) | よくある質問 | テキスト: `質問があります` |
| F(下段・右) | お店について | リンク: `https://sakuraspa-osaka.com/ja` |

※ B・C・Eは、テキスト受信をトリガーに応答メッセージ(自動応答)で給与表や
よくある質問を自動返信する設定にすると、深夜の問い合わせにも対応できて取りこぼしが減る。
※ 求人ページをサイトに実装したら、B・C・Eをリンクに差し替えてもOK。

# メンズエステ総合求人アカウント用リッチメニュー(VELOUR SPA + SAKURA SPA 共通)

## ファイル

| ファイル | 内容 |
|---|---|
| `richmenu-recruit-all.png` | 総合求人用画像(2500×1686px、950KB) |
| `richmenu-recruit-all.html` | デザイン元データ |

両店共通の求人窓口アカウント「メンズエステ総合求人」用。
下段に店舗ボタン(VELOUR SPA=ダーク / SAKURA SPA=ピンク)を配置し、
各店の詳細を見てから応募できる導線。

## 各ボタンのアクション設定(推奨)

| 位置 | ボタン | アクション |
|---|---|---|
| A(上段・左) | 応募する | テキスト: `応募希望です` |
| B(上段・中) | 給与・報酬 | テキスト: `給与・報酬について教えてください` |
| C(上段・右) | 体験入店 | テキスト: `体験入店を希望します` |
| D(下段・左) | VELOUR SPA | リンク: `https://velourspa-nippombashi.com/recruit` |
| E(下段・中) | SAKURA SPA | リンク: `https://sakuraspa-osaka.com/ja` ※求人ページ実装後に差し替え |
| F(下段・右) | よくある質問 | テキスト: `質問があります` |

※ 応募・体入のテキストが届いたら「ご希望の店舗(①ヴェロアスパ/②サクラスパ/③どちらでも)を
教えてください」と返す運用にすると、どちらの店への応募か迷わず振り分けられる。
応答メッセージ(自動応答)に登録しておけば深夜も自動で一次対応できる。

## 画像を修正して再書き出しする方法

`richmenu.html` を編集後、PowerShellで:

```powershell
& "C:\Program Files\Google\Chrome\Application\chrome.exe" --headless=new --disable-gpu --force-device-scale-factor=1 --window-size=2500,1686 --hide-scrollbars --screenshot="出力先\richmenu.png" "file:///HTMLのパス"
```

※ 出力先・HTMLパスは日本語を含まないフォルダ(例: `C:\Temp`)にコピーしてから実行すること(日本語パスだと失敗する)。
